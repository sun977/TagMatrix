## 项目上下文

### 技术栈
- 编程语言：TypeScript / Go (Wails)
- 框架版本：Vue 3 + Element Plus + Vue Router + Pinia
- 样式预处理器：SCSS

### 现有架构理解
- 架构模式：SPA (单页面应用)，由于是 Wails 项目，没有浏览器的地址栏。
- 核心模块：
  - `Layout.vue`: 整体布局（侧边栏、顶部导航、主内容区）。
  - `main.scss`: 全局样式与 Element Plus 变量覆盖。
  - `views/*`: 对应的各个功能页面。
- 设计变化点：
  - 布局：左侧导航栏背景变为浅灰色 (`#f7f7f9`)，占据整高；右侧内容区域独立，顶部不再有贯穿全局的 Header。
  - 右下角增加悬浮的 AI 助手入口。
  - 主页面顶部区域包含标题与全局任务状态（胶囊形绿底状态栏）。
  - 全局设置变为模态框。
  - 各个页面的细节需要对应设计图进行像素级还原（如圆角、留白、卡片阴影、表格样式等）。
