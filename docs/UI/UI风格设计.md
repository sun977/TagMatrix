## 使用了 skill ui-ux-pro-max

### ui-ux-pro-max 推荐的风格（我选了 极客暗黑风）
🎨 风格选项 1：极客暗黑风 (Dark Mode / Developer Centric) —— ✨ 引擎首推
这是专门针对带有代码编辑器和数据表格的页面推荐的风格，非常具有极客和专业开发感。
主色调：深石板蓝 #0F172A 与 深邃黑 #020617 作为背景，减少长时间盯着屏幕的视觉疲劳。
强调色：明亮绿 #22C55E 或 荧光蓝，用于执行按钮（如“运行SQL”）、成功提示。
字体：Fira Code（等宽代码字体，用于SQL编辑器和表名） + Fira Sans（正文），透出浓厚的技术、精确的氛围。
特效特点：表格行悬停高亮、简洁的边框线条、工具提示（Tooltip）响应迅速。
🎨 风格选项 2：极简瑞士风 (Minimalism & Swiss Style)
如果您喜欢干净、克制、高对比度的后台界面（类似 GitHub 或 Vercel 的控制台），这个风格最合适。
主色调：纯白 #FFFFFF 面板配上极浅的灰背景 #F9FAFB。
强调色：纯黑 #000000 按钮，配合深灰色文字，只在关键告警（如危险操作）使用一点点红色。
字体：Inter 或 Roboto，极为工整、可读性极高。
特效特点：几乎没有阴影，仅用 1px 的精致边框来分割区域，追求极致的整洁和高数据密度，没有多余的修饰。
🎨 风格选项 3：现代 SaaS / 软性 UI (Soft UI Evolution)
如果您希望界面看起来不那么“硬核”，而是更像一个现代、友好的 B2B 商业产品。
主色调：页面背景为偏暖的灰白 #F8FAFC，面板为纯白。
强调色：经典的主题蓝 #3B82F6 或 靛蓝 #6366F1，传达安全、稳定感。
字体：Plus Jakarta Sans 或 System UI，圆润且现代。
特效特点：卡片带有柔和的阴影（Soft shadows）、平滑的圆角（border-radius: 8px 或 12px），交互状态（Hover）带有平滑的渐变过渡（200ms）。

- 极客暗黑风：太黑了，不太行
